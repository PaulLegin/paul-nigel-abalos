# Paul Nigel S. Abalos Portfolio Website

A professional Bootstrap portfolio website for Dr. Paul Nigel S. Abalos, DIT.

## How to use

1. Replace the profile photo:
   - Put your real photo here: `assets/img/profile.jpg`
   - Keep the same filename to avoid editing the code.

2. Update contact details in `index.html`:
   - Replace `your-email@example.com` with your email.
   - Replace `https://github.com/your-github-username` with your GitHub profile.

3. Existing links already included:
   - LinkedIn: https://www.linkedin.com/in/paulnigelabalos/
   - ORCID: https://orcid.org/0000-0003-1866-9274
   - La Mesa: https://lamesa.unaux.com

4. Upload all files to a GitHub repository.
5. Enable GitHub Pages from the repository settings.

## Built With

- Bootstrap 5.3
- Bootstrap Icons
- HTML5, CSS3, JavaScript
